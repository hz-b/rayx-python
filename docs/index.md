# rayx-python

**rayx-python** is a Python API for [RAYX](https://pypi.org/project/rayx/), the ray-tracing engine for synchrotron optics developed at Helmholtz-Zentrum Berlin. It lets you load beamline definitions from `.rml` files, manipulate optical elements and sources, run ray traces, and work with the resulting ray data as numpy arrays or pandas DataFrames.

:::{note}
For a higher-level interface that integrates rayx-python into a full simulation and optimization workflow, see [raypyng](https://raypyng.readthedocs.io/en/latest/).
:::
